## Usage

```
Generate image with DALL-E and print it

Usage: tnap [OPTIONS]

Options:
  -p, --prompt <PROMPT>  Prompt to pass to DALL-E
  -a, --ascii            Convert an image to ASCII art
  -h, --help             Print help
  -V, --version          Print version
```

```sh
$ cargo run 
```
